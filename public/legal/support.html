NomadNote Support

NomadNote is a local-first travel planner. Most issues can be resolved on-device because your data is stored locally.

Common fixes

  • If the app does not load, refresh the page or restart the app.
  • If a map link is not detected, save it as a note and add the pin manually.
  • If you are moving devices, export your trip as JSON and import it on the new device.
  • If you want to reset everything, use the clear local data action in settings.

Before reporting a bug

Please include your device, browser or iOS version, what you were trying to do, and whether the issue happens with demo data.

Contact
