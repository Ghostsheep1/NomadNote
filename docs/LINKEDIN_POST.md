# LinkedIn Project Post Draft

I built and published NomadNote v1.1, a local-first travel planner for people who save too many places and need a trip that will actually feel good in real life.

The differentiator is Trip Stress Radar: instead of only helping users add more places, NomadNote detects when a trip is becoming unrealistic. It scores overload, missing map pins, rain risk, FOMO load, and neighborhood spread, then suggests one clear rescue move.

What I built:

- Next.js 16, TypeScript, Tailwind, Zustand
- Dexie + IndexedDB for local-first storage
- MapLibre + OpenStreetMap for free maps
- Capture inbox for map links, social links, coordinates, and notes
- Transparent heuristic itinerary builder
- Responsive phone/tablet/desktop UI
- PWA support and Capacitor iOS packaging path
- GitHub Actions CI, App Store metadata, privacy/support docs

The product principle: plan enough to feel confident, not so much that the trip becomes homework.
